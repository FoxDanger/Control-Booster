===== How to use these functions =====

1 - You have to setup these functions in some mapping software with OSC Support.

2 - Setup the IP for the Local IP 127.0.0.1 if you have this option, otherwise this is already default for softwares that doesn't has the IP option.

3 - Setup and open the door 7002.

4 - Copy the function for whatever button or knob you want. You can have the same function in more than on place at the same time.

===== Some tips about functions =====

Tip 1: Jog wheels functions can be setup on Knobs.

Tip 2: Trackball functions can be setup on Knobs. Just setup the X function for a knob and the Y function for another knob.

Tip 3: Knob functions can be setup on jog wheels, but most of knob functions has a reset feature when you press the knob. So if you setup a knob function for a Jog wheel (that can't be pressed) you gonna lose the reset feature. So you have to live with that or create a new reset function (button) on the code and setup the new function for some button on your controller.

Tip 4: If your controller has knobs without that can't be pressed, than you gonna lose the feature as well. Same solution for the jogs so...

Tip 5: The Jog/Knob sensitivity are sugestions based on Tangent Panels knobs and jogs. They probably can be different in other MIDI Controllers. Test and use as you like it.

Tip 6: As you use my code, you still need use the Davinci Shortcuts map that I provide with TWB 2 on your Davinci, otherwise, some functions that use non-default shortcuts, will not work correctly.
